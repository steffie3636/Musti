## 🚀 Deployment: GitHub + Vercel

### Schritt 1: GitHub Repository erstellen

1. Gehe zu [github.com](https://github.com) → **New repository**
2. Name: `musti-kanada` (oder beliebig)
3. Visibility: **Private** (nur für Musti sichtbar) oder Public
4. **Ohne** README erstellen (wir pushen selbst)

### Schritt 2: Dateien hochladen

**Option A – Direkt im Browser (einfachste Methode):**
1. Repository öffnen → **Add file** → **Upload files**
2. Alle 4 Dateien per Drag & Drop hochladen
3. **Commit changes** klicken

**Option B – Via Terminal (für Techies):**
```bash
cd musti-reise
git init
git add .
git commit -m "🍁 Mustis Kanada-Reise"
git remote add origin https://github.com/DEIN-NAME/musti-kanada.git
git push -u origin main
```

### Schritt 3: Vercel verbinden

1. Gehe zu [vercel.com](https://vercel.com) → Kostenlos anmelden (mit GitHub)
2. **New Project** → GitHub Repository auswählen: `musti-kanada`
3. Einstellungen so lassen wie sie sind → **Deploy** klicken
4. ✅ Fertig! Vercel gibt dir eine URL wie:
   `https://musti-kanada.vercel.app`

### Schritt 4: Custom Domain (optional)

Falls du eine persönliche URL möchtest (z.B. `musti70.vercel.app`):
- Vercel → Dein Projekt → **Settings** → **Domains**
- Wunsch-Domain eingeben → Vercel prüft Verfügbarkeit
